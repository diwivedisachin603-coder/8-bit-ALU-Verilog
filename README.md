🔥 8-bit ALU Design in Verilog

Overview

This project implements an 8-bit Arithmetic Logic Unit (ALU) using Verilog HDL.
It supports arithmetic and logical operations with flag generation.

Operations Supported

- Addition
- Subtraction
- Multiplication
- Division
- AND, OR, NOT
- NAND, NOR
- XOR, XNOR

Features

- 16-bit output for multiplication
- Overflow detection
- Zero flag generation
- Division-by-zero handling

Tools Used

- Verilog HDL
- Xilinx Vivado

Project Structure

- src/ → Design code
- tb/ → Testbench
- docs/ → Simulation waveform

Future Scope

- 32-bit ALU
- Pipelined ALU
- FPGA implementation
